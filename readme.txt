Algorithms coding exercises

Thank you for your interest in a job at Cost Engineering Consultancy! We would like you to make some coding exercises to determine your general programming skill. 

The exercises are provided are as a Maven project, but please solve the exercises uses Java 8 only without adding extra dependencies.
Try to limit the time you spent per exercise to 30 minutes. 

Please complete the exercises in order of difficulty:
1) Jump counter
2) Best shuffle
3) Chain breaker

The code can be found in src/main/java folder with package eu.costengineering.interview.algorithms. Each problem also has an example test in the src/test/java folder, but feel free to add you own tests!
